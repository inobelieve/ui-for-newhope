# ui-for-newhope

> this is a UI project for New Hope Group company's business

## 使用方法

- 安装依赖
``` bash
# npm install ui-for-newhope --save
```

- 在main.js中全量引入
```js
import {nhUI} from 'ui-for-newhope'
Vue.use(nhUI);
```

- 按需引入
```js
import {nhButton} from 'ui-for-newhope'
Vue.use(nhButton);
```
